Universal Interface Core

A lightweight, highly portable notification and modal engine designed for standard web applications and rapid game UI loops.

Installation Architecture

Thanks to our auto-injector, you don't even need to link CSS files! Just initialize the unified orchestration script at your baseline layer:

<script type="module" src="./main.js"></script>

Once linked, the global ui namespace is instantly available across your entire application.

Toast Alerts

Non-intrusive notification banners that stack and clear gracefully.

Standard Syntax Blueprint

ui.toast.show({
  message: "Operation completed successfully!",
  type: "success",
  theme: "glass",
  duration: 4000
});

Popup Modals

Screen-blocking transaction nodes that capture layered user forms and button actions.

Standard Syntax Blueprint

ui.popup.show({
  title: "Security Clearance",
  message: "Verify your administrative authorization handles.",
  theme: "stealth",
  inputs: [
    { type: "text", name: "user", label: "Operator ID", placeholder: "e.g. Master_1" }
  ],
  buttons: ["GRANT", "DENY"],
  onSubmit: (index, data) => {
    if (index === 1) console.log("Authorized access for: " + data.user);
  }
});
